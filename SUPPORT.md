# Support

Builder Journal documents the public state, direction, and maintenance posture of the ecosystem.

## Where To Ask

- Use GitHub Issues for corrections, broken links, documentation gaps, or ecosystem-status questions.
- For project-specific support, open an issue in the affected repository.
- For security concerns, follow [SECURITY.md](SECURITY.md).

## Maintainer Expectations

This is an independent project. Support is best-effort and focused on documentation accuracy, public project status, and maintainability.
