# Contributing

Builder Journal is the public command center for your software ecosystem. Contributions should improve project records, maintenance clarity, standards alignment, or public documentation.

## Useful Contributions

- Fix stale links, typos, or unclear project references.
- Improve ecosystem maps, status notes, maintenance checklists, or decision records.
- Suggest documentation that makes repository health easier to audit.
- Open issues for inconsistencies across public repositories.

## Process

1. Open a focused issue or pull request.
2. Keep changes factual, restrained, and easy to review.
3. Do not add private implementation details, secrets, credentials, or proprietary operational notes.
4. Follow the standards in [ecosystem-standards](https://github.com/sparshsam/ecosystem-standards).
